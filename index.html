The Development of an Address Book and a Cash Register Using JavaScript

Cash Register

This is the development of the web project 1 containing the cash register using JavaScript.Various functions were created in each script. These include:

  1. An Add function: this is used to add the different item cost.
  2. A scan function :was used to add each item and the amount of quantity inside the cash register being purchased by the different customers.
  3. The transaction method was developed to calculate the charges or bill for each customer based on the amount of products purchased.

Address Book

These are the various functions and their descriptions of what they are used for.

  1. A add function was also created in the address book to add the different contacts.
  2. A search function which was used to iterate through the list of contacts in the address book.
